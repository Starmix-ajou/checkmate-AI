# 예시 1: 최소 입력
기능 이름: 회원 가입


# 예시 2: 기본 설명 추가
기능 이름: 댓글 작성  
설명: 사용자가 게시글에 댓글을 남길 수 있어야 해요.


# 예시 3: 기능 목적 포함
기능 이름: 할 일 추가  
설명: 사용자가 해야 할 일을 등록할 수 있어야 해요.  
목적: 작업을 목록으로 정리해서 관리하기 위해 필요해요.


# 예시 4: 사용 시나리오 포함
기능 이름: 이메일 알림 설정  
설명: 사용자가 특정 이벤트에 대해 이메일 알림을 받을 수 있어야 해요.  
목적: 중요한 이벤트를 놓치지 않도록 알림을 받아야 하기 때문이에요.  
시나리오: 사용자가 설정 페이지에 들어가서 이메일 알림을 켜면, 이후 특정 조건이 충족되면 이메일을 받아요.


# 예시 5: 상세 정보 거의 모두 입력
기능 이름: 비밀번호 재설정  
설명: 사용자가 비밀번호를 잊었을 경우 이메일을 통해 재설정할 수 있어야 해요.  
목적: 사용자 계정 접근이 불가능할 경우를 대비하기 위함이에요.  
시나리오: 로그인 화면에서 '비밀번호를 잊으셨나요?'를 클릭한 후 이메일을 입력하면, 인증 메일을 받아서 새 비밀번호를 설정할 수 있어요.  
입력값: 이메일 주소, 새 비밀번호  
출력값: 비밀번호 변경 성공 메시지  
전제 조건: 사용자가 이미 계정을 가지고 있어야 함
